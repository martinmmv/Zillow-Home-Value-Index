# Zillow-Home-Value-Index
Exploring the Zillow Home Value Index (ZHVI), a measure of the typical home value and market changes across a given region and housing type. It reflects the typical value for homes in the 35th to 65th percentile range. Available as a smoothed, seasonally adjusted measure and as a raw measure.
Used the following Python libraries to clean, transform, explore, and visualize ZHVI data:
1. pandas
2. numpy
3. matplotlib
4. seaborn
